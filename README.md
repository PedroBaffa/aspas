# 🌪️ Aspas
> *"Destaque o que importa."*

O **Aspas** é uma plataforma leve, intuitiva e acessível para gestão de tarefas ágeis (Issue-Tracking), desenvolvida como uma alternativa de melhor custo-benefício e usabilidade a sistemas complexos do mercado, como o Jira.

## 🎯 Objetivo do Projeto
Resolver dores comuns em gestão de equipes:
- **Preço:** Licenças caras em softwares tradicionais.
- **Usabilidade:** Curvas de aprendizado longas e interfaces não amigáveis.
- **Performance:** Sistemas pesados e lentos.
